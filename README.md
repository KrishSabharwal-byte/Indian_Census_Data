# 🇮🇳 Indian Census Data Analysis & Visualization

An interactive data analytics project built using **Indian Census Data** to explore demographic, educational, and infrastructure trends across India through maps, charts, and dashboards.

---

# ✨ Features

* 📊 Interactive Visualizations
* 🗺️ Geographic & State-wise Analysis
* 📈 Population & Literacy Insights
* ⚖️ Sex Ratio Analysis
* ⚡ Electricity & Infrastructure Data
* 🎓 Higher Education Statistics
* 🔍 Multi-parameter Filtering Dashboard

---

# 📌 Parameters Used

* 👥 Population
* 📚 Literacy Rate
* ⚖️ Sex Ratio
* 🎓 Higher Education
* ⚡ Electricity Availability
* 🌆 Urban/Rural Distribution
* 📍 Population Density

---

# 🛠️ Tech Stack

* 🐍 Python
* 🐼 Pandas
* 📊 Plotly
* 🎛️ Streamlit
* 📉 Matplotlib & Seaborn

---

# ⚙️ Workflow

1. 📥 Data Collection
2. 🧹 Data Cleaning
3. 🔎 Exploratory Data Analysis
4. 📊 Visualization & Mapping
5. 🚀 Dashboard Development
